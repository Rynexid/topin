# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in TOPIN, please report it privately by opening a security advisory on GitHub or contacting the maintainers directly.

Please do not report security vulnerabilities through public GitHub issues.

## What to Include

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

## Response

We will acknowledge your report within 48 hours and provide a timeline for the fix. We appreciate your help in keeping TOPIN secure.
